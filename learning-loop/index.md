# Learning Loop

This section provides a holistic reflection on learning across the MSc Artificial Intelligence programme.

## Purpose
The Learning Loop is intended to:

- connect learning across modules
- identify recurring themes and concepts
- link theory with practical application
- review progress against professional development goals
- build an integrated view of my academic and professional growth

## Reflection Approach
The Learning Loop will capture:

- key insights gained
- connections between modules
- application in professional practice
- areas requiring deeper understanding
- long-term development goals

## Programme Integration
As the programme progresses, this section will serve as a consolidated view of how individual module learnings combine into broader capability development in Artificial Intelligence.
